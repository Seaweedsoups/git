Directive * s-dropdown

=ng-model - null

下拉框值


=ng-disabled - false

禁用控件


=data - null

为数组时候为本地数据, function时返回$.Deferred


@text-key - "text"

text取值


@value-key - "value"

value取值, 为false时候, 取值json对象


@nothing - "Please Select"

默认显示内容


@close-on-select - true

选中后收起下拉列表


@multiple - false

是否可多选


@required - false

设置为true后, 必须有一个选中项


@auto-width - false

宽度对齐


&on-select - null

选中时触发


@type - "click"

"click"或"mouseover"

Constructor

$( selector ).dropdown( options )

selector - 选择器或 DOM 元素

options - 参数设置

$( ".ui.dropdown" ).dropdown();
Options

data - null
