Directive * s-tree

=root-ids - null

Root节点ID


@parent-key - "parentId"

关联键


@text-key - "text"

text取值


@value-key - "value"

text取值


=data - null

为数组时候为本地数据, function时返回$.Deferred


@collapsed - true

为false时展开所有节点


@closeSameLevel - false

展开节点时关闭同一级的节点


@placeholder - "Type to search..."

The same to input placeholder


&onSelect - null

选中节点时回调


=controller - null

Tree's controller, See Method

Constructor

$( selector ).ntree( [options] )

selector - 选择器或 DOM 元素

options - 参数设置

$( ".ui.ntree" ).ntree( { data: function() { var deferred = $.Deferred(); $.ajax( { url: "/src/demo/tree/tree.json", dataType: "json" } ) .done( function( data ) { data = data.result.catalog; deferred.resolveWith( JSON.parse( data ) ); } ); return deferred.promise(); }, parentKey: "a03_parent_id", textKey: "a03_name", valueKey: "a03_id", rootIds: [ "C000000000481935" ] } );


Options

rootIds - null

Root节点ID


parentKey - "parentId"

关联键


textKey - "text"

text取值


value - "value"

text取值


data - null

为数组时候为本地数据, function时返回$.Deferred


collapsed - true

为false时展开所有节点


onSelect - null

选中节点时回调

$( ".ui.ntree" ).ntree( { onSelect: function( e, item, dataContext, level ) { console.log( arguments ); } } );

duration - 150

展开动画速度, 单位毫秒


closeSameLevel - false

展开节点时关闭同一级的节点


selector4content - ".content"

树形节点容器选择器


selector4filter - "input[name=filter]"

过滤框选择器


filter - null

function或hash对象, 作用于每一级节点的过滤器

$( ".ui.ntree" ).ntree( { filter: { "0": function( item ) { return !!item.value; }, "1": function( item ) { return item.value > 100; } } } );

formatter -

节点格式化

$( ".ui.ntree" ).ntree( { formatter: function() { return "<p style='padding-left: " + ((level - 1) * 2) + "em;'><i class='icon'></i><span>" + item[ settings.textKey ] + "</span></p>"; } } );

tree.collapsed( nodeid )
