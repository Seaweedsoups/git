AngularJS
Markup
<s-pagination index="index" max="20"></s-pagination>
<div class="ui pagination flat"> <div class="content"></div> </div>
Javascript
angular .module( "demo.pagination", [ "$ui.pagination" ] ) .controller( "paginationController", [ "$scope", function( $scope ) { $scope.index = 7; } ] );
Directive * s-pagination

=index - 1

当前页数


=max - 1

最大页数


API
HTML
<div class="ui pagination"> <div class="content"></div> <span class="normal">跳转至</span> <input type="text" maxleng="4"> <span name="go">GO</span> </div>


Constructor

$( selector ).pagination( [options] )

selector - 选择器或 DOM 元素

options - 参数设置

$( ".ui.pagination" ).calendar();


Options

max - 1

最大页数


index - 1

当前页数


onPageChange - null

页数变动时触发


selector4content - ".content"

Selector of the pagination container


selector4input - "input:text"

Selector of the input box


selector4button - "[name=go]"

Selector of the button




Method

pagination.val( [ value ] )

value - Number

设置/取值 当前页数
